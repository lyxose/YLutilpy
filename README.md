# YLutilpy
Some handy functions for Python data analysis

**Usage:**
1. by installing
    ```bat
    git clone https://github.com/lyxose/YLutilpy.git
    pip install ./YLutilpy
    ```
2. by import source code
    Move the `utility` folder to the current work dir, then
    ```python
    from utility import *
    ```
